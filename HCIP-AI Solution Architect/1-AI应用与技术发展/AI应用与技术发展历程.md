# AI技术发展历程
## 由小到大
- 算法，参数量膨胀，单位由$Million$ $\to$ $Billion$
- 数据:训练数据增加，单个模型训练数据集可多达万亿$token$
- 算力：算力规模提升至$EFLOPs$
```ad-tip
title:FLOPs
$FLOPs$ 是指的是每秒浮点运算次数。$E$代表的是一百京，所以$EFLOPs$称为每秒一百京次($10^{18}$)浮点运算
```
## 学习算法
- 基于规则：基于固定的规则推理，不具备学习能力，根据关键词匹配
- 机器学习：通过数据驱动，自动优化算法模型，依赖人工特征工程
	- $SVN,\ KN N,\ K-means,\ GBDT$等，处理聚类，回归，分类等任务
- 深度学习：基于神经网络自动提取特征，擅长处理图像，语音，文本等复杂数据
	- $R esNet,\ VGG,\ GoogleNet,\ Transfom er$等，部分任务已经超过人类
- 大模型：通过海量参数和多任务学习实现通用智能，如文本生成，逻辑推理等
	- $GPT,\ GLM,\ StableDif fusion,\ DeepSe ek$等，深度学习在参数规模和任务泛化上的突破
## 数据集
|     | 数据规模                         | 数据多样性                                         |
| --- | ---------------------------- | --------------------------------------------- |
| 小模型 | 数据集通常在百万到千万级别                | 数据通常针对特定领域，无需广泛覆盖。数据分布更加集中                    |
| 大模型 | 大模型需要处理和学习复杂的关系和模式，因此需要大规模数据 | 数据集涉及多个领域和主题，不局限于单一模态和特定领域，呈现 **多模态**、**跨领域** |
## 大模型vs小模型
| 小模型         | 大模型         |
| ----------- | ----------- |
| 学习能力上限底     | 学习能力强       |
| 不同任务需要不同模型  | 一个模型解决多个任务  |
| 训练数据上限低     | 训练数据上限高     |
| 单一模态数据      | 多模态能力强      |
| Few-shot能力差 | Few-shot能力强 |
